## 2. 전체 기능 목록

### 2.1 인증 및 계정 관리

**AUTH-001: 사용자 인증**
- 이메일 회원가입 및 인증
- 소셜 로그인 (Google OAuth)
- 비밀번호 재설정

**PROF-001: 프로필 관리**
- 프로필 정보 편집
- 언어/시간대 설정
- 계정 삭제

**SUB-001: 구독 관리**
- 플랜 선택 및 변경
- 결제 수단 관리
- 결제 내역 조회
- 구독 취소

---

### 2.2 대시보드

**DASH-001: 메인 대시보드**
- 채널 요약 통계 (총 영상, 댓글, 평균 감성 점수)
- 최근 분석 영상 목록 (최대 5개)
- 주요 지표 카드 (긍정 비율, 답변 필요, 악성 댓글)
- 빠른 분석 입력 (URL 입력 → 즉시 분석)

**DASH-002: 통계 위젯**
- 감성 트렌드 그래프 (7일/30일)
- 활동 히트맵 (요일/시간대별 댓글 활동)
- 인기 키워드 Top 10
- 키워드 클라우드

---

### 2.3 영상 관리

**VIDEO-001: 영상 목록**
- YouTube 채널 연동 (OAuth)
- 영상 목록 자동 동기화
- 영상 정보 표시 (썸네일, 제목, 조회수, 댓글 수, 감성 점수)
- 테이블 뷰 / 카드 뷰 전환
- 정렬 (날짜, 조회수, 댓글 수, 감성 점수)
- 필터링 (날짜 범위, 조회수 범위, 댓글 수 범위, 분석 상태)
- 검색 (영상 제목)
- 페이지네이션 (10/25/50/100)
- 채널목록
- 일괄 분석

**VIDEO-002: 영상 상세**
- 영상 정보 섹션
- 댓글 통계 대시보드
- 탭 네비게이션 (댓글 목록, 감성 분석, 키워드 분석, 토픽 분석)
- 분석 버튼/추가분석버튼 (미분석인경우 분석, 이미 분석 한 경우 최신댓글 추가 분석 및 통계 업데이트)
- YouTube에서 보기 링크

---

### 2.4 댓글 분석

**COMMENT-001: 댓글 수집**
- YouTube Data API 연동
- 영상별 전체 댓글 수집
- 답글(Replies) 포함 수집
- 좋아요/작성자/작성일 메타데이터 저장
- 수집 진행 상태 표시
- 증분 수집 (신규 댓글만)

**ANALYSIS-001: AI 감성 분석**
- GPT-4 기반 감성 분석 (긍정/중립/부정)
- 감성 점수 산출 (1-5점)
- 감성별 댓글 분류
- 감성 분포 시각화 (도넛 차트)
- 시계열 감성 변화 그래프
- 영상별 감성 비교 (바 차트)
- 가장 긍정/부정 댓글 Top 5

**ANALYSIS-002: 키워드 분석**
- TF-IDF 기반 키워드 추출
- 워드클라우드 시각화 (인터랙티브)
- 키워드 순위 테이블
- 키워드별 댓글 필터링
- 키워드 트렌드 그래프
- 키워드 비교 (최대 5개)

**ANALYSIS-003: 토픽 분석** [STARTER+]
- LDA 기반 토픽 모델링
- 토픽별 댓글 클러스터링
- 토픽 시각화 (네트워크 그래프)
- 토픽 라벨 자동 생성
- 토픽별 대표 댓글 샘플

**ANALYSIS-004: 댓글 자동 분류** [STARTER+]
- AI 기반 카테고리 분류 (질문, 피드백, 칭찬, 비판, 스팸)
- 카테고리별 댓글 필터링
- 답변 필요 댓글 하이라이트
- 크리에이터 멘션 댓글 추출
- 카테고리별 통계

---

### 2.5 시청자 인사이트 [PRO+]

**AUDIENCE-001: 시청자 분석**
- 총 고유 댓글 작성자 수
- 평균 댓글 작성 빈도
- 활동 시간대 히트맵
- 가장 많이 댓글 작성한 이용자 순위
- 충성 팬 vs 신규 시청자 세그먼트
- 각 그룹 특징 분석

**AUDIENCE-002: 시청자 요청사항**
- AI 자동 추출 요청사항 목록
- 카테고리별 분류 (콘텐츠 주제, 편집 스타일, 기능 요청)
- 언급 횟수 집계
- 우선순위 점수 산출
- 요청 트렌드 분석

**AUDIENCE-003: 시청자 페르소나**
- AI 기반 시청자 그룹 생성 (최대 5개)
- 페르소나별 특징 분석
- 비율 및 분포
- 대표 댓글 샘플

---

### 2.6 고급 분석 [PRO+]

**ANALYTICS-001: 비교 분석**
- 영상 간 비교 (최대 5개)
- 기간별 비교
- 경쟁 채널 비교 (공개 데이터)
- 비교 지표: 댓글 수, 감성 점수, 키워드, 반응 패턴
- 레이더 차트, 병렬 바 차트

**ANALYTICS-002: 성과 분석**
- 영상별 종합 성과 점수 (0-100)
- 댓글 참여율 분석
- 시간대별 참여도 그래프
- 영상 길이 vs 참여도 상관관계
- 댓글 감성 vs 조회수 상관관계

**ANALYTICS-003: 트렌드 분석**
- 월별/주별 주요 지표 변화
- 인기 상승/하락 키워드
- 새로운 토픽 발견
- 예측 분석 (다음 달 예상 댓글 수, 감성 트렌드)

---

### 2.7 AI 어시스턴트 [PRO+]

**AI-001: 답변 생성**
- GPT-4 기반 답변 자동 생성
- 답변 톤 선택 (친근한, 전문적인, 유머러스한)
- 답변 템플릿 저장 및 관리
- 사용량 표시 및 제한 (PRO: 100회/월, ENTERPRISE: 무제한)
- 생성된 답변 편집 가능

**AI-002: 콘텐츠 추천**
- 다음 콘텐츠 주제 추천 (최대 10개)
- 추천 이유 및 예상 시청자 반응
- 관련 키워드 제시
- 업로드 최적 시간대 제안
- 요일별/시간대별 추천 히트맵

**AI-003: 인사이트 챗봇**
- 자연어 질의응답
- 데이터 기반 인사이트 제공
- 관련 차트/데이터 자동 표시

---

### 2.8 악성 댓글 관리 [PRO+]

**MOD-001: 악성 댓글 감지**
- AI 기반 악성 댓글 자동 감지
- 감지 유형: 혐오 표현, 스팸, 트롤
- 위험도 평가 (높음/중간/낮음)
- 악성 댓글 목록 및 필터링

---

### 2.9 리포트

**REPORT-001: 자동 리포트**
- 주간/월간 리포트 자동 생성
- 리포트 스케줄 설정
- 이메일 발송 옵션
- PDF 다운로드
- 리포트 내역 조회

**REPORT-002: 커스텀 리포트** [PRO+]
- 리포트 생성 마법사 (기간, 포함 항목, 템플릿, 형식 선택)
- 템플릿: 표준, 상세, 요약, 브랜드용
- 내보내기 형식: PDF, PowerPoint, Excel
- 저장된 템플릿 관리

---

### 2.10 연동 관리

**INTEG-001: YouTube 연동**
- OAuth 인증 및 재인증
- 채널 정보 표시
- 권한 상태 확인
- 연동 해제
---

### 2.11 설정

**SETTING-001: 계정 설정**
- 프로필 정보 편집
- 비밀번호 변경

**SETTING-002: 구독 관리**
- 현재 플랜 정보
- 플랜 변경
- 결제 내역
- 결제 수단 관리
- 구독 취소

**SETTING-003: 데이터 관리**
- 데이터 내보내기 (CSV, JSON)

- 데이터 보유 정책

**SETTING-004: 환경 설정**
- 언어 설정

**SETTING-005: 보안**
